emb-loop is a common platform for AI to autonomously develop embedded software.  

It separates the CLI, runtime, and protocol, providing the same development experience across various microcontrollers.


emb-loop は AI が組み込みソフトウェアを自律開発するための共通プラットフォームである。

CLI、Runtime、Protocol を分離し、様々なマイコン上で同一の開発体験を提供する。
